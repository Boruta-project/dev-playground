DevPlayground

Embeddable developer widgets (quotes, console, ASCII micro-animations). Use with Google Sites via iframe.

  • Random Quote Widget — embed this URL in an iframe.
  • Dev Console Widget — login and enjoy the random output.
  • API endpoints: /api/quotes/index.json, /api/console/messages.json, /api/console/ascii.json

To embed in Google Sites: Insert → Embed → By URL → paste the widget URL (example: https://yourusername.github.io/dev-playground/widgets/random-quote.html)
